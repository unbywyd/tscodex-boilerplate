# כרטיס מוצר

![כרטיס מוצר]({{BASE_URL}}/uploads/docs/images/component-product-card.png)

## סקירה כללית

כרטיס המוצר הוא רכיב UI המציג תוכנית תמחור יחידה עם כל המידע הרלוונטי. הכרטיס כולל שם התוכנית, תיאור, מחיר, רשימת תכונות וכפתור לבחירת התוכנית.

## תיאור

כרטיס המוצר מציג תוכנית תמחור בודדת בפורמט נקי ומקצועי. הכרטיס תומך בהדגשת תוכניות פופולריות ומאפשר למשתמשים לבחור תוכנית בקלות.

## תכונות

| תכונה | תיאור |
|-------|-------|
| שם ותיאור מוצר | כותרת ותיאור קצר של התוכנית |
| מחיר עם פורמט מטבע | הצגת מחיר בפורמט מטבע מקצועי (USD, EUR, RUB) |
| רשימת תכונות | רשימת תכונות עם סימני V |
| תג פופולרי | אינדיקטור ויזואלי לתוכניות פופולריות |
| כפתור בחירת תוכנית | כפתור פעולה לבחירת התוכנית |

## Props

| Prop | Type | תיאור |
|------|------|-------|
| product | Product | אובייקט נתוני המוצר |
| onSelect | (id: number) => void | פונקציית callback לבחירת התוכנית |

## עיצוב

הכרטיס כולל:
- **Hover Effect:** אפקט צל מוגבר בעת מעבר עכבר
- **Popular Badge:** גבול וצל מוגברים לתוכניות פופולריות
- **Responsive:** תמיכה מלאה במכשירים שונים

## ישויות קשורות

הרכיב קשור לישות הבאה:
- **Product** - ישות המוצר המכילה את כל המידע על התוכנית

## דפים קשורים

הרכיב מופיע בדף הבא:
- **Products Page** - דף המוצרים הראשי
