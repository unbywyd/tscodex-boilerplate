# Product Card Component

## Overview

The Product Card component displays individual pricing plans with their features and selection options. It is used throughout the Products page to present each plan in a consistent, visually appealing format.

![Product Card]({{BASE_URL}}/uploads/docs/images/component-product-card.png)

## Component Details

**Name:** Product Card  
**ID:** `product-card`  
**Description:** Displays product/pricing plan with features list and selection button

## Features

- Product name and description
- Price with currency formatting
- Features checklist with checkmark icons
- Popular badge indicator (optional)
- Select plan button with icon

## Props

| Prop | Type | Description |
|------|------|-------------|
| product | Product | Product data object containing name, description, price, and features |
| onSelect | (id: number) => void | Callback function triggered when user selects a plan |

## Styling

### Visual States

- **Default:** Standard card with border and shadow
- **Popular:** Enhanced border (`border-primary`) and shadow (`shadow-md`) with "Popular" badge at the top
- **Hover:** Elevated shadow (`shadow-lg`) for better interactivity

### Responsive Design

The component is fully responsive and adapts to different screen sizes:
- Text sizes adjust for mobile, tablet, and desktop
- Spacing and padding scale appropriately
- Button sizes optimize for touch targets on mobile devices

### Interactive Elements

- **Hover Effect:** Smooth shadow transition on card hover
- **Button States:** Primary button for popular plans, secondary for others
- **Badge Display:** "Popular" badge appears only on featured plans

## Usage Example

The Product Card is used within the Products page grid to display each pricing tier. Multiple instances can be rendered side-by-side, with one typically marked as "popular" to guide user selection.
