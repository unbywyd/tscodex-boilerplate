# Products Page

## Overview

The Products page displays a catalog of pricing plans in a responsive grid layout. Users can browse different product tiers, compare features, and select plans.

![Products Page]({{BASE_URL}}/uploads/docs/images/screen-products.png)

## Page Details

**Name:** Products Page  
**Path:** `/prototype/products`  
**Description:** Browse product catalog with pricing plans

## Features

- Display products in responsive grid
- Show pricing with currency formatting
- Highlight popular products
- List product features
- Select plan action

## Implementation

- **Component:** `ProductsPage`
- **File:** `src/prototype/pages/ProductsPage.tsx`
- **Status:** Implemented

## Layout

The page uses a responsive grid layout that adapts to different screen sizes:
- **Mobile:** Single column layout
- **Tablet:** Two columns
- **Desktop:** Three columns

Each product is displayed using the Product Card component, which provides consistent styling and interaction patterns across all plans.
