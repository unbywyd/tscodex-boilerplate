# דף מוצרים

![דף מוצרים]({{BASE_URL}}/uploads/docs/images/screen-products.png)

## סקירה כללית

דף המוצרים מאפשר למשתמשים לעיין בקטלוג המוצרים ותוכניות התמחור של המערכת. הדף מציג את כל התוכניות הזמינות בתצוגת רשת רספונסיבית, תוך הדגשת תוכניות פופולריות והצגת תכונות מפורטות לכל תוכנית.

## תיאור

דף המוצרים הוא עמוד קטלוג מרכזי המאפשר למשתמשים:
- לעיין בתוכניות התמחור השונות
- להשוות בין התוכניות השונות
- לבחור תוכנית מתאימה

הדף מציג את המוצרים בתצוגת רשת רספונסיבית התואמת למכשירים שונים.

## תכונות עיקריות

| תכונה | תיאור |
|-------|-------|
| תצוגת רשת רספונסיבית | המוצרים מוצגים ברשת התואמת למסכים שונים |
| עיצוב מחירים עם מטבע | הצגת מחירים בפורמט מטבע מקצועי |
| הדגשת מוצרים פופולריים | תוכניות פופולריות מסומנות בבירור |
| רשימת תכונות | כל תוכנית כוללת רשימת תכונות מפורטת |
| פעולת בחירת תוכנית | כפתור לבחירת התוכנית בכל כרטיס |

## רכיבים קשורים

הדף משתמש ברכיב הבא:
- **Product Card** - כרטיס מוצר להצגת כל תוכנית

## ישויות קשורות

הדף קשור לישות הבאה:
- **Product** - ישות המוצר המכילה את כל המידע על התוכניות
